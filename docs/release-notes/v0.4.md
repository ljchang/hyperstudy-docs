# Version 0.4.x Release Notes

## v0.4.16 (November 2025)

### Fixed
- Set correct Firebase Auth claims based on experimenter approval status
- Sync Firebase Auth custom claims when approving users

## v0.4.15 (November 2025)

### Fixed
- Fix automatic release notification from GitHub Actions

## v0.4.14 (November 2025)

### Added
- Participant rejoin and disconnect timeout system
- Comprehensive notification system with security hardening and automated release integration

### Fixed
- Fix notification creation during index building
- Update notification UI styling and improve error handling
- Fix experiment visibility websocket events and add recruitment:updated notification
- Fix NotificationManager styling to match admin dashboard dark theme

## v0.4.0 - v0.4.13

These versions included incremental improvements and bug fixes during the v0.4 development cycle. Key features introduced:

### Core Features
- Enhanced experiment state management
- Improved video synchronization stability
- LiveKit reconnection handling improvements
- Host reconnection logic refinements

### Data & API
- V3 Data API with comprehensive data types
- Enhanced data export functionality
- API key management system

### User Experience
- Improved experiment runner UI
- Enhanced participant waiting room experience
- Better error handling and user feedback

### Infrastructure
- Kubernetes deployment improvements
- Horizontal scaling support
- Enhanced monitoring and logging
